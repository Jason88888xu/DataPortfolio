Project Overview (Introduction)¶

  • This experimental model uses machine learning algorithms to predict future stock prices of the U.S. stock market.
  • This model is only applicable to the US stock market and macroeconomic data (such as indices, industries, and commodities).
  • This experimental model is using the Random Forest model, a machine learning algorithm, for the stock price predictions.

Disclaimer: The stock price predictions generated by this Python analysis should not be used for actual market trading. This Python stock price prediction model is intended solely for self-learning purposes and does not provide any investment advice.

Process the Date column¶

Since the results predict the next N days, we need to delete the oldest N dates and insert the newest N working days.

In [57]:
# get today's date
today = datetime.date.today()

# 创建一个空的DataFrame
new_dates = pd.DataFrame(columns=['Date'])

# Add the N dates
for _ in range(predict_period):
    today += datetime.timedelta(days=1)
    # check to see if it is weekday，if not, keep going find the next weekday
    while today.weekday() >= 5:  # 5 is saturday，6 is sunday
        today += datetime.timedelta(days=1)
    new_dates = pd.concat([new_dates, pd.DataFrame({'Date': [today]})], ignore_index=True)
